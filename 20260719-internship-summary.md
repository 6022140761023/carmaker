## 二、项目概述

本项目为校园卡丁车的电驱系统开发，核心任务是设计并实现一块基于 **ESP32 + TMI8720PD** 的直流电机驱动板，驱动两台 **RS-220 直流有刷电机**，实现校园低速载人工具的动力系统。

**技术架构：**

```
电源 12V/2A ──┬──→ 驱动板 U2（12V 输入）──→ Q1/Q2 功率级 ──→ U3 ──→ 电机
              │
              └──→ buck 降压模块 ──→ ESP32 (5V)
                                  GPIO25/26 → U4 IN1/IN2（3.3V 逻辑）
                                  GND → U4-3（必须共地）
```
