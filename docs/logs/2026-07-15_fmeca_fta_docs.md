## 2. 下午：电驱专项 FMECA/FTA 文档

按老师要求，任务 B（FMECA）和任务 C（FTA）合并为一个**电驱专项精简文档**，聚焦电驱组实际完成的三件事：

### 核心内容

| 实际工作 | 对应失效模式 | RPN | 状态 |
|---|---|---|---|
| 电机转速测试（2/4/8/12V 分档） | 电机转速异常 | 48 | 已验证 |
| 线缆过细 → 换 0.8mm² | 线缆过热/烧毁 | 105 | **已解决** |
| 去除转接口 → 直接焊接 | 接触不良/断路 | 108 | **已解决** |

### 文档结构

1. **电驱系统概述**：驱动板 + 电机 + 线缆三模块，突出"直接焊接"设计决策
2. **FMECA 分析**：4 项失效模式，2 项高风险（线缆过细、接触不良）标绿"已解决"
3. **FTA 故障树**：顶事件"电驱系统失效"，线缆断路、端子接触不良标注"已解决：改为直接焊接"
4. **已解决失效案例**：两个完整案例（现象→根因→措施→验证）
5. **电机转速测试数据**：2/4/8/12V 分档实测表
6. **结论与建议**：两项高风险闭环，3 条后续建议

### 技术要点

- 线缆选型：0.2mm² 信号线（载流 0.5A）→ 0.8mm² 动力线（载流 3~5A），余量 3 倍
- 焊接工艺：电机端直接焊接（350°C，2~3 秒），焊点拉力测试 5N/3s
- 测试方法：分电压点测试，空载电流 0.1~0.5A，验证电机特性与线缆载流
