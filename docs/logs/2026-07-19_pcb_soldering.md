## 1. PCB焊接

### 1.1 SMT贴片手焊
- 对 U1（TMI8720PD，DFN-10-EP）和 Q1/Q2（AGM210MAP，PDFN-8）进行热风手焊
- 使用热风返修台 300~350°C，焊锡膏辅助
- 注意芯片摆放方向和引脚对齐，焊后目视检查 EP 焊盘吃锡情况

### 1.2 插件元件焊接
- 焊接 F1（PPTC 保险丝 H60-250）、C1/C2/C3（电解电容）、R1/R2（贴片电阻）
- U2/U3（MKDS 接线端子）、U4（1×6P 排针）
- **特别注意电解电容正负极方向**（长脚为正，壳体带灰色条为负）
- 保险丝、端子焊牢，保证机械强度和过流能力

## 2. 双电机驱动方案讨论

### 方案A：并联驱动（放弃）
- 将两个后轮电机并联到同一块板子的 U3 端子
- 优点：省一块板子，接线简单
- 缺点：两轮强制同速，转向时内外轮差速需求无法满足；对整车结构强度和转向机构要求极高，方案激进
- **结论：放弃**

### 方案B：双板独立驱动（采用）
- 使用两块电驱板，各驱动一个后轮电机
- 每块板独立接 12V 电源和 U3 电机输出
- 两块板的 U4-3（GND）与 ESP32 共地
- 电控预留两个 GPIO 控制第二块板子的 IN1/IN2

```
ESP32 GPIO25 ──→ 板1 IN1  ──→ 左轮电机
ESP32 GPIO26 ──→ 板1 IN2
ESP32 GPIO27 ──→ 板2 IN1  ──→ 右轮电机
ESP32 GPIO28 ──→ 板2 IN2
ESP32 GND    ──→ 板1 U4-3 ──→ 板2 U4-3（共地）
12V 母线     ──→ 板1 U2-1 ──→ 板2 U2-1
```

- **结论：采用此方案**，可控性强，支持差速调速和原地转向
