# 开发日志：全车串联调试与电阻发错应急方案（2026-07-20）

**日期**：2026-07-20
**阶段**：全车系统集成调试
**项目**：ESP32 + TMI8720PD 直流电机驱动板（RS-220，12V）

## 1. 全车串联调试

今日进行整车各模块的串联调试，包括：
- 车架结构拼装完成
- 电驱模块、电控模块、电源模块物理拼接
- 各模块间走线布置（电机线、电源线、信号线）

**集成方式**：采用面包板集成各个功能模块，便于调试和快速修改连接。

## 2. 电驱板状态

### 已确认正常
- 板子本身焊接无误，无短路/虚焊
- U1（TMI8720PD）、Q1/Q2（AGM210MAP）工作正常
- U2/U3 端子、U4 排针连接可靠
- 走线布局合理

### 问题发现
- 厂家发货错误：**R1/R2 贴片电阻发成 1000Ω（应为 100Ω）**
- 后果：U3 输出电压降至原来的 1/10（约 1.2V，正常应为 12V × 占空比）
- 原因分析：1000Ω 限流电阻导致 U1 输入驱动电流不足（仅 3.3mA），Q1/Q2 栅极驱动电压不够，MOS 管半导通

## 3. 应急方案

### 短期措施
- **采用 BTS（电池直接供电/备用驱动方案）临时驱动电机**，保证整车调试不中断
- BTS 方案：12V 电池直接给电机供电，通过简单开关或继电器控制通断，绕过电驱板的 PWM 调速

### 中期措施
- **已购买正确规格电阻（100Ω 1206 贴片）**
- 选择顺丰发货，争取最短时间内到货
- 到货后立即更换 R1/R2，恢复电驱板正常 PWM 调速功能

## 4. 风险评估

| 风险 | 影响 | 缓解措施 |
|---|---|---|
| BTS 无 PWM 调速 | 电机只有全速/停止，无法精细控制 | 短期可接受，先用开关控制 |
| 顺丰延迟 | 电驱板恢复时间推后 | 已选最快物流，备选京东次日达 |
| BTS 电流冲击 | 直接启动电流大，可能影响电池/电机 | 限流电阻或软启动电路临时接入 |

## 5. 下一步

- 等 100Ω 电阻到货，热风枪更换 R1/R2
- 更换后验证 U3 输出电压恢复正常（12V × 占空比）
- 从 BTS 切换回电驱板 PWM 调速
- 完成整车电控联调（双板驱动、差速转向、速度控制）

## 6. 经验沉淀

- **元件到货必须验货**：贴片电阻等小件容易被发错，到货后万用表量一下阻值再焊
- **备用方案思维**：核心元件有问题时，要有 B 方案保证项目不卡死
- **物流时间纳入计划**：关键元件补货要预留物流时间，选最快的快递
- **面包板集成优势**：调试阶段用面包板方便快速改线，确认稳定后再固定焊接
