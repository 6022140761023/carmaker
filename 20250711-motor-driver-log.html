💡 核心设计理念：模块化底板

底板采用极简设计——不焊接核心控制芯片，只画插座和接口：

  • ESP32 开发板通过 2×15 排母 插接，可随时拆换
  • 两路驱动电路独立模块化：每路集成 TMI8720 驱动芯片 + 两颗 AGM210MAP MOSFET
  • 底板仅提供：电源输入（XT60）、滤波电路、接口插座、电机接线端子、电流采样电阻
  • 所有 GPIO 信号通过排母引出，便于调试与复用

这种模块化设计使得原型阶段可快速迭代：驱动模块损坏只需更换对应模块，无需重新打板。

2.2 电路架构说明

每路电机驱动采用 TMI8720 + 双AGM210MAP 组成经典H桥拓扑：

  • TMI8720 作为预驱动芯片，接收 ESP32 的 PWM 和方向控制信号，输出栅极驱动电平
  • 两颗 AGM210MAP 分别作为 H 桥的高侧和低侧功率管，由 TMI8720 的栅极驱动引脚控制通断
  • 通过 PWM 占空比调节电机转速，通过 H 桥切换实现正反转

2.5 原理图绘制要点

  • 功能分区布局：电源区（XT60 + 滤波）→ 驱动区（TMI8720 + AGM210MAP H桥）→ 控制区（ESP32 排母接口）
  • 网络标签规范：使用 MOTOR1_PWM、MOTOR1_DIR、MOTOR1_EN 等语义化标签，便于后期调试
  • 电源走线：电源主回路设置 50mil 线宽，确保承载 5A 以上电机电流
  • 信号走线：PWM 信号线采用 10mil 线宽，远离大电流回路以减少电磁干扰
  • GND 铺铜：底层整面铺铜，降低地回路阻抗，提升EMC性能

3.1 为什么选择 TMI8720 + AGM210MAP 方案？

结论：综合考虑校园低速载人工具的实际功率需求（~100W级）、项目预算限制（学生自费）、以及工程实践的学习目标，TMI8720 + AGM210MAP 芯片级方案是最优解——它在性能、成本、教育意义三者之间取得了良好平衡。

3.2 安全考虑

🛡 硬件级保护

TMI8720 内置过流保护（OCP）和过温保护（TSD），当电流超过阈值或芯片温度过高时自动关断输出，无需软件干预即可保护电路安全。同时，使能引脚（EN）由 ESP32 独立控制，可在检测到异常时立即拉低，实现 μs 级响应切断。

🔌 电源保护

输入端采用 XT60 防反插连接器，物理上杜绝电源接反风险。电源入口并联大容量电解电容 + 瓷片电容，吸收电机启停和换向时产生的尖峰电压，保护 MOSFET 免受过压击穿。

📊 电流监控

每路电机串入 0.01Ω 采样电阻，经差分运放放大后送入 ESP32 ADC。软件中设置电流阈值，当检测到堵转或短路电流时自动停止 PWM 输出并记录故障日志，形成"硬件保护 + 软件监控"的双重安全防线。

3.3 包容性设计（适合校园载人低速工具）

为什么不是高速大功率方案？ 我们设计的目标用户是校园中的每一位成员——赶课的学生、推着手推车的后勤工人、在校园漫步的老人和儿童。低速、安全、可靠是首要原则。TMI8720 + AGM210MAP 方案将最高车速控制在 12-15km/h 的安全范围，避免了因追求高性能而带来的安全隐患。

同时，芯片级方案的成本优势使原型更加亲民——如果方案成功，未来可以以更低的成本推广到更多校园，让更多需要便捷出行的人受益。这不仅是一项技术设计，更是一种对社会包容性的主动回应。

四、遇到的问题与解决方案

⚠️
问题1：原理图连线时 TMI8720 的栅极驱动引脚与 AGM210MAP 的匹配

TMI8720 的输出驱动电流有限，直接驱动 AGM210MAP（输入电容较大）可能导致开关速度过慢，增加开关损耗。

✅
解决方案：查阅 TMI8720 数据手册，确认其栅极驱动电流为 1.5A（源）和 2.0A（漏），足以驱动 AGM210MAP 的 ~3nC 栅极充电电荷。为优化开关速度，在栅极串联 10Ω 电阻限制充放电电流，并在栅源极间并联 10kΩ 下拉电阻防止浮空导通。
⚠️
问题2：电源滤波电容容值选择——能否有效抑制电机启动浪涌？

12V 电机启动瞬间电流可达额定电流的 3-5 倍（约 5-8A），如果滤波电容容值不足，母线电压会跌落至 TMI8720 欠压保护阈值以下。

✅
解决方案：计算所需储能：假设电压跌落不超过 0.5V，浪涌持续时间 10ms，所需电容 C = I·Δt / ΔV = 8A × 10ms / 0.5V = 160,000μF。这个容值过于庞大不现实。实际采用工程折衷：输入端选用 470μF 电解电容，配合 TMI8720 的软启动功能（内置1ms斜坡启动），将浪涌电流控制在 2A 以内，同时软件中实现"渐进式加速"策略——PWM 占空比在 0.5 秒内逐渐升高，避免瞬时大电流冲击。
⚠️
问题3：电流采样电路共模干扰

采样电阻位于 H 桥低侧，电机工作时采样电阻两端的共模电压在 0-12V 之间快速跳变，普通运放难以准确放大差模信号。

✅
解决方案：选用高共模抑制比（CMRR > 100dB）的差分运放（如 INA180），并严格遵循"开尔文连接"法则——采样电阻的两根引线分别直接连接到运放输入端，避免大电流路径上的压降干扰。同时在运放输出端增加 RC 低通滤波（截止频率 ~1kHz），滤除 PWM 开关噪声。

六、实习总结与体会

今日是技术密集且成果丰富的一天。从上午学习立创EDA到下午完成基于 TMI8720 + AGM210MAP 双路H桥驱动底板的方案设计，我经历了一次完整的硬件设计决策链路，收获如下：

  • 工程决策需要多维度权衡：TMI8720 + AGM210MAP 方案不是参数最"强"的，但它是在功率能力、成本控制、易用性和学习价值之间最优的平衡点。好的工程师不是在追求极端性能，而是在约束条件中找到最佳解。
  • 安全设计必须前置：从 TMI8720 的过流保护到采样电阻的电流监控，再到软件中的渐进式加速策略，安全不是事后补救，而应贯穿设计的每一个环节。这既是工程责任，也是伦理底线。
  • 从"用模块"到"做芯片级设计"的跨越：相比于直接购买现成的驱动模块，自己用 TMI8720 + AGM210MAP 搭建 H 桥让我真正理解了 MOSFET 栅极驱动、死区时间、续流路径等之前仅停留在书本上的概念。这大概就是工程实践的意义——让知识不再是纸上的符号，而是手中掂量的真实。
