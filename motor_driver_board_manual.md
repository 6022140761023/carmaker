# TMI8720 直流电机驱动板 V1.0 使用说明

> 方案：TMI8720PD（全桥栅极驱动）+ AGM210MAP ×2（P+N 功率管）
> 适用电机：12V DC 有刷电机（RS-220 等）
> 主控：ESP32（外接，杜邦线连接）

---

## 一、板子概览

| 项目 | 参数 |
|---|---|
| 输入电压 | **DC 12V（严禁超过 20V，禁止接 24V）** |
| 最大持续电流 | 理论 25A（受 PCB 散热限制，RS-220 实际 <2A，余量充足） |
| 控制方式 | 2 路 PWM（IN1/IN2），逻辑电平 1.8/3.3/5V 兼容 |
| PWM 频率 | 推荐 10~20kHz |
| 使能方式 | 无专用 EN 脚，IN1/IN2 双低持续 1ms 自动休眠 |
| 保护 | F1 自恢复保险丝（电源输入）+ TMI8720 内部保护 |

## 二、对外接口定义

### U2 —— 电源输入（MKDS 螺钉端子）

| 脚位 | 信号 | 接什么 |
|---|---|---|
| U2-1 | +12V | 直流电源正极 / 12V电池正极 |
| U2-2 | GND(−) | 直流电源负极 / 12V电池负极 |

⚠️ 接反会烧 Q1/Q2，上电前用万用表确认极性！

### U3 —— 电机输出（MKDS 螺钉端子）

| 脚位 | 信号 | 接什么 |
|---|---|---|
| U3-1 | OUT1 | 电机一端（RS-220 任意一根线） |
| U3-2 | OUT2 | 电机另一端 |

转向反了？交换电机两根线，或程序里 IN1/IN2 互换即可。

### U4 —— 控制接口（2.54mm 排针 1×6，接 ESP32）

| 脚位 | 信号 | 接 ESP32 | 说明 |
|---|---|---|---|
| U4-1 | IN1 | GPIO 25 | 正转 PWM 输入 |
| U4-2 | IN2 | GPIO 26 | 反转 PWM 输入 |
| U4-3 | GND | GND | **公共地，必须接！** |
| U4-4 | NC | 不接 | 预留空脚 |
| U4-5 | NC | 不接 | 预留空脚 |
| U4-6 | NC | 不接 | 预留空脚 |

## 三、控制逻辑（真值表）

| IN1 | IN2 | 电机状态 |
|---|---|---|
| 0 | 0 | 停止（滑行），1ms 后芯片自动休眠 |
| PWM | 0 | 正转，占空比越大越快 |
| 0 | PWM | 反转，占空比越大越快 |
| 1 | 1 | 刹车（快速停止） |

## 五、测试程序（串口正负数调速）

```cpp
/* TMI8720 串口调速测试
 * 串口输入 -255~255: 正=正转, 负=反转, 0=停, 绝对值=速度
 * Arduino IDE, 开发板选 "ESP32 Dev Module", Core 3.x
 */
#define IN1_PIN 25
#define IN2_PIN 26
#define PWM_FREQ 20000
#define PWM_RES  8

void setMotor(int v) {
  v = constrain(v, -255, 255);
  if (v > 0)      { ledcWrite(IN1_PIN, v);  ledcWrite(IN2_PIN, 0);   }
  else if (v < 0) { ledcWrite(IN1_PIN, 0);  ledcWrite(IN2_PIN, -v);  }
  else            { ledcWrite(IN1_PIN, 0);  ledcWrite(IN2_PIN, 0);   }
}

void setup() {
  Serial.begin(115200);
  ledcAttach(IN1_PIN, PWM_FREQ, PWM_RES);
  ledcAttach(IN2_PIN, PWM_FREQ, PWM_RES);
  setMotor(0);
  Serial.println("输入 -255~255 控制电机");
}

void loop() {
  if (Serial.available()) {
    int v = Serial.parseInt();
    while (Serial.available()) Serial.read();
    setMotor(v);
    Serial.printf("速度设定: %d\n", v);
  }
}
```

## 六、上电测试流程（严格按顺序）

1. **空载检查**：焊完板，万用表量 U2 两端电阻 —— 接近 0Ω = 短路，禁止上电
2. **上电**：直流电源 12V、限流 2A 接 U2，量 VM 母线是否 12V
3. **空载信号验证**：不装电机，ESP32 接好 3 根线，串口输入 150，万用表量 U3 两端应有电压摆动；输入 -150 极性翻转
4. **装电机**：接 RS-220，输入 100 起步，确认转向；逐步加大到 255
5. **反向测试**：输入 -100，确认反转

## 七、常见问题

| 现象 | 原因 | 解决 |
|---|---|---|
| 电机不转 | U4-3 GND 没接 | 检查共地（90%是这个问题） |
| 电机只往一个方向转 | IN1/IN2 有一根虚接 | 重新插拔杜邦线 |
| 转向与预期相反 | 电机线接反 | 交换 U3 两根线，或程序互换 IN1/IN2 |
| 上电保险丝就跳 | VM/GND 短路或电源接反 | 断电，万用表查短路点 |
| 电机抽搐 | 电源限流太小 | 限流放宽到 3A 以上 |
| 转速上不去 | PWM 占空比限制 | 程序里输入更大数值（最大255） |

## 八、安全红线

- ❌ VM 严禁超过 20V（AGM210MAP 耐压 20V，24V 必击穿）
- ❌ 电源严禁接反（无防反接电路）
- ❌ U3 输出严禁短路（虽然芯片有过流保护，但不要考验它）
- ✅ 调试时直流电源限流 2~3A，接错也烧不了东西
